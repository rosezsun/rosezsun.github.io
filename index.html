JAVA零基础-入门

在CSDN上随便找了一入门的JAVA课程，节数还是可以接受的，没有九九八十一节的那种。
1.先讲一些历史：
————三个版本：
J2SE、基础版本，做一些桌面软件 java1.5后，为 java se
J2ME、移动版，现在淘汰了已经，现在是安卓/ios java1.5后，为 java me
J2EE、企业版，可以做一些大型的系统，大型网站 java1.5后，为 java ee，然后2018年改称Jakarta EE

————版本：
1.1 1.2 1.3 1.4 直到java1.5,改动比较大
之后都是JAVA5 JAVA6 JAVA7 JAVA8（主流） …2018年后每半年更新一次

2.介绍JAVA特点：
————跨平台语言，就是给不同的操作系统（平台）写程序，只要写一个就可以了，都是兼容的。一次编写，处处运行。
？为什么？：在操作系统上安装了一个JVM（java的虚拟机），有各种类型的JVM，所以JAVA在JVM上运行就不会存在跨平台了。
因此运行java程序的流程：
开发语言（写代码，java）-》编译（class）-》执行class

3.JVM：java虚拟机
jre相当于：JVM + 核心类库 —只能运行java，但不能开发
jdk相当于：jre + 运行环境工具 —运行+开发。用它就ok
