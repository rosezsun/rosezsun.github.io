函数！被呼噜声吵得睡不着爬起来学习

今天是个好日子啊，想不到我脾气变这么好了。
废话不多说，学习！开始函数部分了。

1.函数可以减少“代码复制”，可以方便移植。
2.函数——接收值，返回值的代码块。

void sum(int beg, int end) //函数头 函数名（参数表）
{
int i; //函数体
int sum=0;
for(i=beg;i<=end; i++)
{
sum+=i;
}
}

调用的时候 函数名（参数）；
3.如果调用的时候不加()?不报错，但是结果不是想要的。
4.return 停止函数执行，并送回一个值。
5.一个函数里面可以有很多return 但是设计理念中的单一出口觉得这样不好。
6. void的函数，可以没有return，有的话也不能有返回值。而且在主函数中调用不能用来做运算（因为都没有返回值了）

7.主函数写在最后，C编译器自上而下分析代码
8.函数如果写在后面，需要在前面进行声明：int sum（int ,int ); 这是函数的原型。而且可以不写参数的名字。

9.传参。传参的数据类型和函数的类型不一样，有的会warnning，像我的根本就不会报。C编译器会偷偷转换。

10.传参是传值。每个函数有自己的变量空间。

11.本地变量？ooo，函数内部的变量。生存时间和作用域。
甚至可以随便拉一堆{}来定义变量，离开这个块，变量就消失了。确实是这样，外面调用i的时候报错了。

  1. int f(void) 和 int f()的区别。后者代表不知道其中是什么参数，前者就是明确的表示没有。后者的话可能会出现传参错误，所以别用了吧，找bug也挺麻烦。

13.C不能函数嵌套定义。
