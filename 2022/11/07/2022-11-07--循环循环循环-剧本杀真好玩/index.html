循环循环循环...剧本杀真好玩~

11月7日，2022.
前两天周末出去玩剧本杀了，电音蝌蚪也到了，但是好难弹好！

今天学习循环。

1.1 while(){} //它的特点是判断之前先循环，循环的时候注意需要包含改变判断条件的动作，不然很可能死循环？
1.2 do{}while(); //注意这个分号，表示do-while结束。先做再判断，这对取第一个边界值有好处。

这个do-while我不复习的话，我都快忘记了。还是需要多写代码多练习。
1.3 for(int i=1;i<n;i++){} 这个int i=1放在for里面，需要在c99下才可以允许。
for(初始条件；循环条件；循环每次要做的事情) 三个表达式可以有缺省。
那这样的话，把上面这三句换成别的东西岂不是也可以，比如把每次要做的事情换成循环体中的东西。那这三个部分都只能是一句话嘛？可以用大括号嘛？欸，不行，编译通过不了。
2.测试程序方法：
2.1纸上自己写；
2.2 边界值；
2.3printf的插入，比如 printf(“*”);

3.新函数！ rand() 随机一个整数，使用%来控制范围。前面加上随机种子srand(time(0)); 具体缘由我现在还不清楚。

4.int变成浮点：1.0*x（^0^）

(感觉那一个算平均数对于与除数count的判断还是需要外加一个if，没有想到怎么在循环或者初值搞定它)
int sum = 0;
int count = 0;
int usr;
scanf(“%d”,&usr);
while(usr != -1){
sum += usr;
count ++;
scanf(“%d”,&usr);
}
