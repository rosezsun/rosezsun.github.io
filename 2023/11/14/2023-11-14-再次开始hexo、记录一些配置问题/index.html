再次开始hexo、记录一些配置问题

很长时间没有用hexo了，这次配置上有一些需要记录的地方，因为发现自己之前都没有过一个配置的记录。

首先，安装git，简单的apt-get 就可以
然后，node.js，官网https://nodejs.org/en/download/，apt-get 也可以，但是不是最新版，最好都是安装最新的。
然后，hexo，使用npm安装，npm install hexo -g
然后，依赖，npm install –save hexo-deployer-git，但是这个依赖要注意，据我自己的经验，在hexo文件夹下运行初始化语句：hexo init，之后，还需要在这个文件夹下再次安装这个依赖。
ssh密钥的配置：cd ~/.ssh
ssh-keygen -t rsa -C “邮件地址”
把密钥复制到github上，.pub上的

hexo init
配置 _config.yml
替换下列语句：
deploy:
type: git
repository: git@github.com:liyunchen/liyunchen.github.io.git
branch: main

然后hexo g
hexo s //看一个本地的预览
hexo d//发布

主要的难点有二：
其一：github登录，这个现在的方法就是站长工具检测下IP，然后改hosts之后重启应用配置。
其二：ERROR Deployer not found: git，这是依赖问题，虽然在外面安装了依赖，但是在hexo init之后，还是要进入到文件夹里再运行一次npm install –save hexo-deployer-git
